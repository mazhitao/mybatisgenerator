本项目使用Mybatis-Plus工具进行自动化生成代码。

该项目与Mybatis-Plus的代码生成器差不多，只是用了自定义的模板。
通过MysqlGenerator可以快速生成Entity、Mapper、Mapper XML、Service、Controller等各个模块的代码，
极大的提升了开发效率。

使用步骤：
一、运行com.mzt.generator.mysql.mybatisplusgenerator.MysqlGenerator.main方法

二、输入模块名和对应的数据库表名

三、即可生成成功。

参考链接：https://mp.baomidou.com/guide/generator.html
